Why are wrangling skills important?

“Classroom data are like teddy bears and real data are like a grizzly bear with salon blood dripping out its mouth” Jenny Byran

tidyr functions

Use the tidyr package to reshape dataframes

gather()

transform from wide to long

dataframe %>% gather(newcolumn, values, columns:to_gather)
spread()

transform from long to wide

dataframe %>% spread(oldcolumntospread, values)
separate()

split a single column into multiple

dataframe %>% separate(columntosplit, names of new columns, sep = "-")
unite()

combine multiple columns into single

dataframe %>% unite(newcolumnmerged, oldcolumns to merge, sep = "-")

dplyr functions

select() select columns of interest
filter() filter rows based on conditions

Rules to filter by:

Less than, greater than, equal to ==, not equal to != etc but also

%in% group membership

is.na is NA

!is.na is not NA

Other subsetting functions

Remove duplicate rows

dataframe %>% distinct()

Select rows by position

dataframe %>% slide(3:5)
group_by() group data by category
summarise() change unit of analysis (mean, sum, count)

Summarise functions: min, max, mean, median, var, sd, but also…

number of values in vector

n()

number of distinct values

n_distinct()
arrange() order the data
mutate() create new variables
join() combine datasets

Each join() function takes arguments x and y (dataframes to join) and by (character vector of variables to join by)

left_join()
right_join()
inner_join()
full_join() #join data retain all values, all rows
semi_join() #keep all rows in x that have a match in y
anti_join() #keep all rows in x that do not have a match in y
